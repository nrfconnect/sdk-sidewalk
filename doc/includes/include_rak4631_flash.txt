Programming with RAK4631 on the nRF52840 DK
===========================================

To program with the `RAK4631`_ module, run the flowing commands instead of ``west flash``:

.. code-block:: console

   pyocd erase -t nrf52840 --chip
   pyocd flash -t nrf52840 build/merged.hex
   pyocd flash -t nrf52840 mfg.hex
   pyocd reset -t nrf52840

The RAK4631 module uses RTT as a terminal.
To view the logs, execute the command:

.. code-block:: console

   pyocd rtt -t nrf52840 -M attach
